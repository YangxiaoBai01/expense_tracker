# Expense Tracker

A web-based expense tracking application built as a class project for students to practice web development fundamentals.

## About

This project is part of **INFO 101: Introduction to Informatics**, a course designed to introduce students to the basics of computing, data, and web technologies. The Expense Tracker serves as a hands-on practice project where students apply what they learn in class to build a functional web application.

> **For educational purposes only.**

## Course Info

| | |
|---|---|
| **Course** | INFO 101 - Introduction to Informatics |
| **Purpose** | Student practice project for web development |
| **Audience** | Beginner-level students, no prior experience required |

## What Students Practice

- HTML/CSS for building and styling web pages
- JavaScript for interactive functionality
- Python (Flask) for backend development
- Handling form data and user input
- Basic CRUD operations (Create, Read, Update, Delete)

## Getting Started

> Make sure you have **Python 3** installed. You can download it at https://www.python.org/downloads/

1. Clone the repository:
   ```bash
   git clone https://github.com/YangxiaoBai01/expense_tracker.git
   cd expense_tracker
   ```

2. (Recommended) Create a virtual environment:
   ```bash
   python -m venv venv
   ```
   Activate it:
   - **Mac/Linux:** `source venv/bin/activate`
   - **Windows:** `venv\Scripts\activate`

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:
   ```bash
   python app.py
   ```

5. Open your browser and go to `http://localhost:5000`
